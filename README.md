# FlameGuard 

Key Features
- Real-time Monitoring : Track device status and safety levels instantly.
- Power Analytics : Detailed graphs for energy usage and fire risk thresholds.
- Smart Control: Remote power cutoff and scheduling for connected appliances.
- Shared Access : Manage household safety by sharing device control with group members.
- Elegant UI : Premium dark mode support and sleek interactive charts.

 Getting Started
1. Prerequisites : Ensure you have Flutter installed (`flutter doctor`).
2. Setup: Run `flutter pub get` to install dependencies.
3. Execution : Run `flutter run` on your preferred emulator or device.

Tech Stack
- Framework : Flutter (Dart)
- State Management : Provider
- UI Components : custom-built widgets, fl_chart, Google Fonts (Outfit)
- Persistence : shared_preferences
